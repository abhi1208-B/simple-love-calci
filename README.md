# 🌿 Nature Photo Scroller

A beautiful, responsive website for scrolling through stunning nature photography. Built with pure HTML, CSS, and JavaScript.

## Features

✨ **Smooth Transitions** - Elegant slide animations between photos
🎨 **Modern Design** - Beautiful gradient backgrounds and smooth UI
📱 **Responsive** - Works perfectly on desktop, tablet, and mobile devices
⌨️ **Keyboard Navigation** - Use arrow keys to navigate photos
🖼️ **High-Quality Images** - Curated collection of stunning nature photos
📝 **Photo Descriptions** - Each photo includes a title and detailed description

## How to Use

1. **Open the website** - Simply open `index.html` in your web browser
2. **Navigate photos**:
   - Click **Next** or **Previous** buttons
   - Use **arrow keys** on your keyboard
3. **View details** - Read the photo title and description below the image

## Customization

### Add More Photos

Edit the `photos` array in `script.js`:

```javascript
const photos = [
    {
        title: 'Your Photo Title',
        description: 'Your photo description',
        url: 'your-image-url'
    },
    // ... more photos
];
```

### Change Colors

Modify the gradient colors in `styles.css`:

```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```
